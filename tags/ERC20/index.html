    以太坊Solidity开发入门ERC20发币

    2018-11-11 23:46:00

    使用ERC20的开源代码 部署到本地测试链/公测链 项目所有代码（在erc20文件夹） 环境部署 安装 geth（以太坊客户端） 安装 truffle（智能合约开发工具包，用于编译与自动部署） 安装 ganache-cli （简易的本地以太坊测试链服务器） 安装命令1234$ brew tap ethereum/ethereum$ brew install ethereum$ npm install -g truffle$ npm install -g ganache-cli
